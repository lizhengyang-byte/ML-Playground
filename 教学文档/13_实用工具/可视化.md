## 概述

数据可视化帮助理解数据分布、发现模式、展示结果。matplotlib 是基础，seaborn 是高级封装。

## 关键代码解释

```python
import matplotlib.pyplot as plt
import seaborn as sns
sns.heatmap(confusion_matrix, annot=True)
sns.pairplot(df, hue="target")
```

## 注意事项

1. 图表要有标题、坐标轴标签、图例
2. 颜色选择要考虑色盲友好
3. 保存图片用 `savefig(dpi=300)`
