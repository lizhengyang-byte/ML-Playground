# 实验日志与追踪：管理你的 ML 实验

> 所属模块：13_实用工具 | 源文件：实验日志与追踪.py | 核心功能：MLflow、实验记录、参数/指标追踪

## 概述

ML 实验管理需要记录每次实验的参数、指标、模型。MLflow 是最流行的开源工具。

## 关键代码解释

```python
import mlflow
mlflow.log_param("learning_rate", 0.01)
mlflow.log_metric("accuracy", 0.95)
mlflow.sklearn.log_model(model, "model")
```

## 注意事项

1. 从第一个实验就开始记录
2. 记录足够信息以便复现
3. 定期清理不好的实验
