# 数据加载：ML 流程的起点

> 所属模块：13_实用工具 | 源文件：数据加载.py | 核心功能：sklearn 数据集、CSV 加载、DataLoader

## 概述

数据加载是 ML 流程的第一步。本文件演示了从 sklearn 内置数据集、CSV 文件、PyTorch DataLoader 等多种数据加载方式。

## 关键代码解释

```python
from sklearn.datasets import load_iris, make_classification
from torch.utils.data import DataLoader, TensorDataset
import pandas as pd
df = pd.read_csv("data.csv")
```

## 注意事项

1. 大数据集不能全部加载到内存，需要分批
2. 数据类型检查很重要
3. 路径问题在不同操作系统上可能不同

## 延伸思考

- **HuggingFace Datasets**：统一的数据集加载库
- **WebDataset**：大规模数据的流式加载
- **内存映射**：mmap 处理超大文件
