﻿## 数学原理

### 1. 数据集的数学表示

机器学习数据集表示为矩阵-标签对：

$$D = \{(x_i, y_i)\}_{i=1}^{n}, \quad X \in \mathbb{R}^{n \times d}, \quad y \in \mathbb{R}^n \text{ 或 } \{0,\ldots,C-1\}^n$$

- $n$：样本数（行数）
- $d$：特征数（列数）
- $C$：分类任务的类别数

### 2. 训练/测试划分

随机划分保持数据独立同分布（i.i.d.）：

$$D = D_{train} \cup D_{test}, \quad D_{train} \cap D_{test} = \emptyset$$

常见比例：80% 训练、20% 测试。分层抽样（`stratify=y`）保持类别比例：

$$\frac{|y=k \cap D_{test}|}{|D_{test}|} \approx \frac{|y=k|}{n}$$

### 3. 数据检查的统计量

| 统计量 | 公式 | 用途 |
|--------|------|------|
| 均值 | $\bar{x}_j = \frac{1}{n}\sum_i x_{ij}$ | 中心趋势 |
| 标准差 | $\sigma_j = \sqrt{\frac{1}{n}\sum(x_{ij}-\bar{x}_j)^2}$ | 离散程度 |
| 偏度 | $\text{skew} = \frac{E[(X-\mu)^3]}{\sigma^3}$ | 分布对称性 |
| 峰度 | $\text{kurt} = \frac{E[(X-\mu)^4]}{\sigma^4} - 3$ | 分布尾部厚度 |

### 4. 代码与数学的对应

| 代码 | 数学含义 |
|------|----------|
| `train_test_split(X, y, test_size=0.3)` | $D \to D_{train} \cup D_{test}$ |
| `stratify=y` | 分层抽样，保持 $P(y=c)$ |
| `iris.data.shape` | $(n, d)$ 数据矩阵形状 |
| `np.bincount(y)` | 各类别样本数统计 |
| `df.describe()` | 均值、标准差、分位数等统计量 |
